# 💰 Loan Approval Predictor

[![Python](https://img.shields.io/badge/Python-3.8+-3776AB?style=for-the-badge&logo=python&logoColor=white)](https://www.python.org/)
[![Scikit-Learn](https://img.shields.io/badge/ML-Scikit--Learn-F7931E?style=for-the-badge&logo=scikit-learn&logoColor=white)](https://scikit-learn.org/)
[![Pandas](https://img.shields.io/badge/Data-Pandas-150458?style=for-the-badge&logo=pandas&logoColor=white)](https://pandas.pydata.org/)

A machine learning project aimed at automating the loan eligibility process. This model predicts whether a client will be approved for a loan based on various financial and personal indicators like credit history, income, and education level.

## 📊 Project Scope
- **Classification Problem**: Predicts "Approved" or "Rejected".
- **Data Preprocessing**: Handling missing values, scaling features, and encoding categorical variables.
- **Model Training**: comparison of various classification algorithms to find the most accurate predictor.

## 🛠 Tech Stack
- **Languages**: Python
- **Libraries**: Scikit-learn, Pandas, NumPy, Matplotlib
- **Tooling**: Jupyter Notebook

## 📂 Repository Content
- `Loan_approval.ipynb`: The complete end-to-end data science workflow from cleaning to model evaluation.

## 🚀 How to Run
1. Clone the repository.
2. Run the `Loan_approval.ipynb` using Jupyter Notebook or Google Colab.
3. Observe the model metrics and feature importance analysis.
